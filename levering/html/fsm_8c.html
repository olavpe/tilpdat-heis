void fsm ( )

The functions main responsibilites is managing the finite state machine. Before entering a switch that manages the state machine it polls:

  • The order buttons
  • floor sensors
  • stop button

The state machine manages the states in the state_t. The behaviour between the states is described in the state diagram and the sequence diagram.

[out]fsm_stateElevator state
